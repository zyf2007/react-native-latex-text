# react-native-latex-text
React Native/Expo LaTeX Formula Rendering Component, supports mixed typesetting of text and formulas, custom formula delimiters, automatic calculation of formula line height, based on MathJax SVG, built-in caching, no native dependencies. | React Native/Expo  Latex 公式渲染组件，支持文字与公式混排/自定义公式分隔符，自动计算公式行高，基于 MathJax SVG，内置缓存，无原生依赖。
